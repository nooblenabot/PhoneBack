SimpleMail

La classe hda cache fourni des fonctions permettant de construire puis d'envoyer des mail teste, html avec ou sans pieces jointes.

Mise en place

Copier le fichier de classe dans un répertoire accessible en inclusion.

  • $mail->addfrom('adressmail' [ , 'nom' ] );

  • $mail->addbcc('adressmail' [ , 'nom' ] );

  • $mail->addreturnpath('adressmail');

$mail = new simplemail;

constructeur des instances de mail. Il initialise les variable de la classe.

$mail->addrecipient('adressmail' [ , 'nom' ] );

Vérifie l'adresse mail puis ajoute un destinataire principal ( To: ). Renouveler l'appel ajoute d'autre destinataire.

$mail->addfrom('adressmail' [ , 'nom' ] );

verifie l'adresse mail puis ajoute l'expediteur (From: ). Renouveler l'appel remplace l'expediteur.

$mail->addcc('adressmail' [ , 'nom' ] );

verifie l'adresse mail puis ajoute un destinataire de copie conforme ( Cc: ). Renouveler l'appel ajoute d'autre destinataire.

$mail->addbcc('adressmail' [ , 'nom' ] );

verifie l'adresse mail puis ajoute un destinataire de copie conforme cachée ( Bcc: ). Renouveler l'appel ajoute d'autre destinataire.

$mail->addsubject('subject');

Ajoute le sujet du mail ( Subject: ). Renouveler l'appel remplace le sujet.

$mail->addreturnpath('adressmail');

verifie l'adresse mail puis ajoute le Return-Path. Renouveler l'appel le remplace.

$mail->addreplyto('adressmail');

verifie l'adresse mail puis ajoute le Reply-To. Renouveler l'appel le remplace.

$mail->addattachement('filename');

Ajoute le fichier en piece jointe.

$mail->addhtmlattachement('filename','cid','content-type');

Ajoute le fichier en piece jointe en vue de l'afficher dans le mail au format html ( comme des image par exemple ). cid ( content-id ) represente le nom auquel vous ferez reference ds votre contenu html ( ex: <img src="cid:monimage1"> ) et content-type precise le type de document ( ex: image/png ).

$mail -> text = 'plain text etc.';

Ajoute le contenu plain-text s'il y a lieu.

$mail -> html ='contenu html';

Ajoute le contenu Html s'il y a lieu.

$mail->sendmail();

Envoie effectivement le mail composé.
